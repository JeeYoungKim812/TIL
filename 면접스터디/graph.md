# Prim 알고리즘

- MST를 구현하는 알고리즘 중 하나
- 시작 정점에서 출발해 신장트리의 집합을 단계적으로 확장하는 방법
- 정점 선택을 기반으로 하는 알고리즘



- ## Prim 알고리즘 동작과정

  - 시작 단계에서는 시작 정점만이 MST 집합에 포함
  - 앞 단계에서 만들어진 MST 집합에 인접한 정점들 중에서 최소 가중치로 연결된 정점을 선택하여 트리를 확장합니다.
  - 위의 과정을 트리가 N-1개의 간선을 가질 때까지 반복합니다.
